### 🌊 About AquaBloom

AquaBloom is a calming and interactive water-tracking app  
designed to help users build healthier hydration habits.

Track your water intake using simple visuals like bottles and glasses  
while enjoying a clean and aesthetic experience 💜

</div>

---

<div align="center">

### ✨ Features

💧 Water intake tracking  
🫧 Minimal & calming UI  
📱 Responsive design  
🌙 Aesthetic purple theme  
📊 Easy progress visualization
